# World Weather Map

The **World Weather Map** is an interactive React-based application designed to provide real-time weather information for locations around the globe.

## Features

- **Interactive Map**: Built using the [MapBox API](https://docs.mapbox.com/api/overview/), the application displays a dynamic map interface.
- **Location Click**: Users can click on any location on the map to view detailed weather information for that area.
- **Weather Info Box**: A pop-up info box shows the weather forecast for the selected location, including temperature, conditions, and more.
- **Search Functionality**: A search box allows users to quickly find specific locations on the map.
- **Temperature Scale Toggle**: Users can switch between Celsius and Fahrenheit temperature scales for convenience.
- **Real Weather Data**: The application integrates with a reliable weather data source, such as the [OpenMeteo API](https://open-meteo.com/en/docs), to provide accurate and up-to-date weather information.

This application combines the power of modern mapping tools with real-time weather data to deliver a seamless and informative user experience.
